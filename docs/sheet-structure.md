# Google Sheet 구조

학교 교직원 교육센터는 학교별 Google Sheet를 운영 데이터 저장소로 사용합니다. 개인정보는 GitHub Pages에 저장하지 않고 학교별 Sheet와 Drive에만 저장합니다.

## 권장 탭

```text
00_학교설정
01_교육목록
02_교직원명단
03_교육대상
04_QR출석기록
05_전자서명기록
06_이수증업로드
07_업로드상태_VIEW
08_최종서명부
99_코드값
```

## 핵심 관계

- `01_교육목록`과 `02_교직원명단`은 직접 연결하지 않습니다.
- 교육별 대상자는 반드시 `03_교육대상`으로 판단합니다.
- 현장 QR 교육의 최종 이수 기준은 `05_전자서명기록`입니다.
- `04_QR출석기록`은 레거시/보조 로그이며 최종 이수 판정 기준이 아닙니다.

## 00_학교설정

필수 설정값:

```text
학교명
교육센터명
학교로고URL
메인컬러
보조컬러
담당부서명
담당자명
담당자 연락처
전자서명 저장 폴더 ID
이수증 저장 폴더 ID
최종 서명부 저장 폴더 ID
개인정보 안내문
adminCode
```

`adminCode`는 관리자 화면 접근 확인에만 사용하며 API 응답에 포함하지 않습니다.

## 01_교육목록

```text
교육ID
교육명
교육일자
교육시간
장소
담당부서
교육구분
QR사용여부
전자서명필요여부
이수증제출필요여부
활성상태
서명부파일ID
서명부파일URL
비고
```

권장 해석:

- `QR사용여부=Y`이면 현장 QR 서명 교육입니다.
- 현장 QR 서명 교육은 `전자서명필요여부`가 비어 있어도 서명 필수로 간주합니다.
- 이수증 업로드 교육은 `이수증제출필요여부=Y`로 관리합니다.

## 02_교직원명단

```text
교직원ID
성명
부서
직위
인증코드
재직상태
권한
비고
```

일반 교직원 화면에서는 인증코드를 요구하지 않습니다. 성명 + 소속부서로 본인을 확인합니다.

## 03_교육대상

```text
교육ID
교직원ID
대상여부
서명제외여부
필수여부
이수증제출대상여부
비고
```

- 현장 QR 서명 교육: 대상자가 QR을 스캔하고 전자서명하면 이수완료입니다.
- 이수증 업로드 교육: `이수증제출대상여부=Y`인 대상자가 이수증을 제출합니다.
- `서명제외여부`는 고급/예외 옵션으로만 사용합니다.

## 04_QR출석기록

레거시/보조 로그 탭입니다. 최종 이수 판정 기준으로 사용하지 않습니다.

## 05_전자서명기록

현장 QR 교육의 출석 및 이수 기준 탭입니다.

```text
서명ID
교육ID
교육명
교직원ID
성명
부서
직책
서명일시
서명파일URL
서명파일ID
저장상태
묶음ID
비고
```

`저장상태`가 `저장완료`, `완료`, `제출`, `승인`이면 이수완료로 판단합니다.
