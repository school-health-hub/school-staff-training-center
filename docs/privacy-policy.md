# 개인정보 저장 원칙

이 프로젝트는 학교별 Google Sheet와 Google Drive를 개인정보 저장소로 사용합니다. GitHub Pages는 정적 화면만 제공하며 개인정보를 저장하지 않습니다.

## 저장 위치

- 교직원 명단: 학교별 Google Sheet `02_교직원명단`
- 교육대상: 학교별 Google Sheet `03_교육대상`
- 전자서명 기록: 학교별 Google Sheet `05_전자서명기록`
- 전자서명 이미지: 학교별 Google Drive 전자서명 폴더
- 이수증 제출 기록: 학교별 Google Sheet `06_이수증업로드`
- 이수증 파일: 학교별 Google Drive 이수증 폴더
- 최종 서명부: 학교별 Google Sheet `08_최종서명부` 및 Google Drive 최종서명부 폴더

## GitHub Pages에 저장하지 않는 정보

- 교직원 명단
- 인증코드
- 관리자 코드
- 전자서명 이미지
- 이수증 파일
- 출석/이수 기록

## 운영 기준

- QR 출석 = 전자서명 출석입니다.
- 현장 QR 교육의 출석 및 이수 기준은 `05_전자서명기록`입니다.
- `04_QR출석기록`은 레거시/보조 로그이며 최종 이수 판정 기준으로 사용하지 않습니다.
- 일반 교직원은 인증코드 없이 성명 + 소속부서로 본인 확인합니다.
- 관리자 화면은 `adminCode`로 보호하되, 코드 원문은 API 응답에 포함하지 않습니다.

## 공개 저장소 주의사항

공개 템플릿 저장소에는 다음 값을 커밋하지 않습니다.

- 실제 Apps Script Web App URL
- Google Drive 폴더 ID
- Google Sheet ID
- 교직원 이름, 부서, 직위 등 실데이터
- 관리자 코드 또는 인증코드
