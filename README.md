# 학교 교직원 교육센터

학교에서 반복 운영되는 교직원 연수, 법정의무교육, 안전교육, 전달연수의 출석·전자서명·이수현황·최종 서명부를 관리하는 공용 템플릿입니다.

## 구조

- Google Sheet: 학교별 교직원 명단, 교육목록, 출석기록 저장
- Apps Script: 시트 읽기/쓰기, 전자서명 저장, 최종 서명부 생성
- Vercel: 교직원용/관리자용 화면 제공

## 개인정보 원칙

교직원 명단, 출석기록, 전자서명 이미지는 학교별 Google Sheet와 Google Drive에만 저장됩니다.
Vercel 서버에는 민감정보를 저장하지 않습니다.
